# TuM2 Mundialista App Icon Pack

Pack completo generado desde el logo mundialista aprobado.

## Dónde usar cada cosa

- App Store promocional: `store/app_store_1024_mundialista.png`
- Play Store promocional: `store/play_store_512_mundialista.png`
- Android legacy: `android/app/src/main/res/mipmap-*`
- Android adaptive: `android/adaptive/`
- iOS: `ios/AppIcon.appiconset/`
- Web favicon/PWA: `web/`
- Flutter launcher icons: `flutter/flutter_launcher_icons_mundialista.yaml`
- Validación visual: `preview/validation_grid_mundialista.png`

## Nota

Esta versión es mundialista/promocional. La versión productiva base de TuM2 sigue siendo el ícono normal.
