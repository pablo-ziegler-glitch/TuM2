# TuM2 App Icon Pack V3

Pack regenerado desde el logo real, sin redibujo manual.

## Dónde usar cada cosa

- App Store: `store/app_store_1024.png`
- Play Store: `store/play_store_512.png`
- Android legacy: `android/app/src/main/res/mipmap-*`
- Android adaptive: `android/adaptive/`
- iOS: `ios/AppIcon.appiconset/`
- Web favicon/PWA: `web/`
- Flutter launcher icons: `flutter/`
- Validación visual: `preview/validation_grid.png`

## Regla

El ícono productivo de PROD es exact-source. Las variantes DEV/STAGING son solo para diferenciar ambientes internos.
