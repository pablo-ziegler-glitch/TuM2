# TuM2 — Assets de íconos en carpeta única plana

Esta carpeta está pensada para IntelliJ / Codex cuando no acepta ZIPs ni estructuras comprimidas.

## Convención de nombres

Todos los archivos están en una sola carpeta:

- Los archivos del ícono productivo base empiezan con `base__`.
- Los archivos del ícono promocional mundialista empiezan con `mundialista__`.

## Regla de uso

- `base__...` = ícono productivo principal.
- `mundialista__...` = variante promocional/eventual, no activa por defecto.
- No redibujar ni reconstruir logos.
- No mezclar esta tarea con TuM2-0013, TuM2-0057 ni TuM2-0026.

## Manifest

Ver `asset_manifest_flat.json` para mapear cada archivo plano contra su ruta original dentro del pack.
